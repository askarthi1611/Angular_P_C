  <h2>Installation</h2>
  <ol>

  </ol>

  <p>Navigate to the project directory:</p>
  <pre><code>cd your-angular-app</code></pre>
  <p>Install dependencies using npm:</p>

  <p>Start the development server:</p>
  <pre><code>ng serve</code></pre>
